# Approach Hold

Selecting an approach hold on the flight plan opens a menu. Changes to the active flight plan take effect immediately.

![Hold Options menu showing Activate Hold, Insert After, Edit Hold, Exit Hold, and Remove options](page_194_image_4_v2.jpg)

* Activate the selected hold
* Insert a waypoint after the hold
* Edit hold details
* Exit the holding pattern before the timer expires
* Remove the selected hold from the flight plan

## ACTIVATE HOLD

![Confirmation dialog for Activate Leg? at EMADE with OK button](page_194_image_2_v2.jpg)

1. Select the hold.
2. Tap **Activate Hold**.
3. Confirm the request.

## REMOVE HOLDING PATTERN

![Confirmation dialog for Remove Holding Pattern? with OK button](page_194_image_3_v2.jpg)

1. Select the hold.
2. Tap **Remove**.
3. Confirm the request.

## EXIT HOLDING PATTERN

![Confirmation dialog for Exit Hold at EMADE? with OK button](page_194_image_1_v2.jpg)

1. Tap **SUSP**.
2. Select the hold.
3. Tap **Exit Hold**.
4. Confirm the request.
