# Flight Plan User Field

**AVAILABLE WITH:**
**GNC 355/355A**

### FEATURE REQUIREMENTS
* Active flight plan for from-to-next route information

![Map display showing user fields with active route identifiers](page_155_image_4_v2.jpg)
