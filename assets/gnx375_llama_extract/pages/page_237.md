# G-AIRMET FILTERS

![Legend of G-AIRMET filters showing colored dashed lines for Freezing Level, Mountain Obscuration, Icing, Surface Winds, IFR, Turbulence, and Low-level Wind Shear.](page_237_image_1_v2.jpg)

**To enable graphical AIRMETs:**

1. Home > **FIS-B Weather** > **Menu**.

2. Toggle **G-AIRMET** on.

3. Tap **Settings**.

4. Select one or more filters.

5. Select between **Current (Auto)** and **All Forecasts**.

6. Return to the FIS-B Weather page.
