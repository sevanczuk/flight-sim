# TRANSPONDER STATUS INDICATIONS

## IDENT

![Transponder display showing XPDR R ID IDENT](page_81_image_1_v2.jpg)

* Reply active
* IDENT function active
* No change to transponder code

## Standby Mode

![Transponder display showing XPDR STBY 1200](page_81_image_3_v2.jpg)

* Standby mode
* Current squawk code (inactive)

## IDENT with New Squawk Code

![Transponder display showing XPDR R ENT+ID](page_81_image_2_v2.jpg)

* Reply active
* Transponder code modified

## Altitude Reporting Mode

![Transponder display showing XPDR ALT 1200 R ID](page_81_image_4_v2.jpg)

* Altitude reporting mode
* Reply active
* Identify function active
* VFR squawk code (active)
