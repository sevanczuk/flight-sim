# Raw Text Reports

![FIS-B Reports menu showing icons for NOTAM, METAR, TAF, Center WX Advisory, SIGMET, PIREP, Winds/Temps Aloft, and Unavailable Products.](page_242_image_1_v2.jpg)

View raw textual data for a FIS-B product.

1. Open the FIS-B WX Status page.

2. Tap **Raw Text Reports**.

3. Select a weather product.

![PIREP icon](page_242_image_3_v2.jpg)

![Raw PIREPs text display showing multiple PIREP reports.](page_242_image_6_v2.jpg)

The latest uploaded data for the product displays on a dedicated page.

![METAR icon](page_242_image_4_v2.jpg)

![Raw METARs list showing locations KAAT, KACV, and KALW with View buttons.](page_242_image_5_v2.jpg)

Multiple reporting stations appear in a list. Tapping **View** displays raw text for the corresponding identifier.

![Unavailable Products icon](page_242_image_2_v2.jpg)

![Raw UNAVAIL text display showing FIS-B service outage reports.](page_242_image_7_v2.jpg)

To view information about products not available due to a service outage, select **Unavailable Products**.
