# Menus

Menus group related controls into an expandable pane, allowing access to multiple functions on a single page. Depending on the number of available functions, a menu may comprise more than one pane.

![Screenshot of a Map Menu with multiple toggle keys like Smart Airspaces, Show Airspaces, Airways, OBST/Wires, TOPO Scale, and Range Ring.](page_25_image_3_v2.jpg)

Multiple panes are accessible by way of a left/right swipe or inner knob turn.

Toggle keys either enable or disable list items. In some cases, **Settings** or **Range** keys provide access to selectable setting options.

![Close-up of the Map Menu active pane indicator showing four dots with the first one highlighted.](page_25_image_4_v2.jpg)

An indicator at the bottom of the menu shows which pane is active.

## POP-UP MENUS

![Screenshot of a pop-up menu for Map Detail with options Low, Medium, High, and Full. Labels point to the underlying page being hidden and the default/current setting.](page_25_image_2_v2.jpg)

Pop-up menus open to the default or previously selected value.

## LISTS

Scrollable lists group control keys related to a single function (e.g., FIS-B Weather). When scrolling, all keys in the list are inactive.

![Screenshot of a scrollable list showing messages like Call FBO, Switch fuel tanks, and Close flight plan with remaining times. A scroll bar is indicated on the right.](page_25_image_1_v2.jpg)
