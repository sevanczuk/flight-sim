# COM Radio Setup

COM radio customization options reside in the System Setup app.

```mermaid
graph TD
    Home --> System
    System --> Setup
```

*Setup options for GNC 355A shown as typical.*

![Screenshot of the System Setup COM radio options on a GNC 355A display, showing Channel Spacing set to 8.33 kHz, Reverse Frequency Lookup, and Sidetone Volume linked to COM VOL.](page_60_image_2_v2.jpg)

For COM radio selections, swipe to the end of the menu.

From here you can:

* Set transceiver channel spacing<sup>1</sup>

* Enable reverse frequency look-up functionality

* Adjust sidetone volume offset

![Diagram showing the menu structure under Setup: Channel Spacing (8.33 kHz, 25 kHz), Reverse Frequency Lookup, and Sidetone Volume (Link to COM VOL, Set offset value).](page_60_image_1_v2.jpg)

<sup>1</sup> GNC 355A only.
