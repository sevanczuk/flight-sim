# Pilot Interface

## Bezel

![Diagram of the Garmin unit bezel with callouts 1 through 6. The screen dimensions are labeled as 1.57" by 4.70".](page_21_image_1_v2.jpg)


<table>
  <tbody>
    <tr>
        <td>1</td>
        <td>**Bezel**<br/>Includes the power key, mechanical knobs, photocell, and SD card slot. Ledges provide hand stability when performing data entry and making selections.</td>
    </tr>
    <tr>
        <td>2</td>
        <td>**Touchscreen**<br/>Multi-touch color display provides controls for unit operation.</td>
    </tr>
    <tr>
        <td>3</td>
        <td>**Photocell**<br/>Measures cockpit ambient light level to automatically adjust display brightness for day and night.</td>
    </tr>
    <tr>
        <td>4</td>
        <td>**SD Card Slot**<br/>Interface for loading database, exporting log files, and updating software. Compatibility with Flight Stream 510 allows wireless database transfer from the Garmin Pilot app via Database Concierge.</td>
    </tr>
    <tr>
        <td>5</td>
        <td>**Power/Home Key**<br/>Powers the unit on or off and provides direct access to the Home page.</td>
    </tr>
    <tr>
        <td>6</td>
        <td>**Inner &amp; Outer Knobs**<br/>Multipurpose dual concentric knob allows data entry, list scrolling, map range control, page navigation, and COM volume and frequency tuning.¹</td>
    </tr>
  </tbody>
</table>

¹ COM is a function of GNC 355/355A only.
