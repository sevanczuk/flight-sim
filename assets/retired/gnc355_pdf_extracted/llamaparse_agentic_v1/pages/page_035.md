

# System at a Glance

## ADC & AHRS

AHRS units have a magnetometer interface for determining magnetic heading.
ADC units have a Pitot-static interface for measuring pressure altitude.

| LRU                             | DISPLAY             | FUNCTION                                                |
| ------------------------------- | ------------------- | ------------------------------------------------------- |
| GDC 74<br/>ADC                  | GNX                 | • Air temperature<br/>• Pressure altitude               |
| G3X<br/>G500/G600               | GPS<br/>GNC<br/>GNX | ADC<br/>• Air temperature<br/>• Pressure altitude       |
| GSU 25/73<br/>Integrated ADAHRS | GNX                 | AHRS<br/>• Heading                                      |
| GRS 77<br/>AHRS                 | GNX                 | • Heading                                               |
| G5/GAD 29D                      | GPS<br/>GNC<br/>GNX | • Heading<br/>• Pressure altitude<br/>• Air temperature |
