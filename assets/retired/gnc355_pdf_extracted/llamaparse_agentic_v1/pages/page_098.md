# Unit Selections

Customize the display of unit settings. Tapping a parameter key opens a menu of the available unit types.

| PARAMETER                  | SETTINGS                                                                                                            |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| \*\*Distance/Speed\*\*     | • Nautical Miles (nm/kt)<br/>• Statute Miles (sm/mph)                                                               |
| \*\*Fuel\*\*               | • Gallons (gal)<br/>• Imperial Gallons (Ig)<br/>• Kilograms (kg)<br/>• Liters (lt)<br/>• Pounds (lb)                |
| \*\*Temperature\*\*        | • Celsius (°C)<br/>• Fahrenheit (°F)                                                                                |
| \*\*NAV Angle\*\*          | • Magnetic (°)<br/>• True (°T)<br/>• User (°U)                                                                      |
| \*\*Magnetic Variation\*\* | • Specify number of degrees for east or west (°E, °W)<br/>• Available only when "User (°U)" is the active NAV angle |


## SPECIFY UNIT TYPE

1.  Review the current unit selections.
2.  Tap the applicable parameter key.
3.  Select a unit type.
