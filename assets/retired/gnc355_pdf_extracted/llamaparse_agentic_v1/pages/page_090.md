Unit customization options allow you to:
- Set the CDI scale
- Display the CDI on-screen³
- Specify runway criteria
- Set the date and time
- Specify COM radio settings¹
- Create shortcuts
- Set the display units
- Adjust display brightness

Other setup options allow you to monitor time in flight and create custom reminder messages. These settings reside in the System Utilities. For details about COM radio settings and Connext Setup options, refer to the respective section.

¹ GNC 355/355A only.
