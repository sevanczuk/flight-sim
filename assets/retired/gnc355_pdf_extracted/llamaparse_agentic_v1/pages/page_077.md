## XPDR Control Panel

Transponder controls are accessible via the **XPDR** key. This key is unavailable when the control panel is active.

XPDR Control Panel Diagram

1. Squawk Code Entry Field
2. VFR Key

**The XPDR key becomes available when you:**
- Enter a squawk code
- Open the XPDR menu
- View a message
- Select the **Mode** key
- Leave the control panel
