**WX DISPLAY ORIENTATION**
The weather app allows you to set the orientation of the weather display. Tap Menu > Orientation, and select from the following options.

| North Up                    | Track Up                                    | Heading Up                                                                         |
| --------------------------- | ------------------------------------------- | ---------------------------------------------------------------------------------- |
| Orients page to True north. | Orients page to current aircraft GPS track. | Orients page to current aircraft heading (requires heading data source interface). |


Page orientation label changes to reflect the selected orientation.
