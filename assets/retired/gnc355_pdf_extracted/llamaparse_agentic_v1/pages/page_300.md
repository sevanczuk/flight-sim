

# Messages

| ADVISORY                                      | CONDITION                                                                                                                                                                                                                                                                                           | CORRECTIVE ACTION                                      |
| --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| Transponder has failed.                       | For GNX 375, or GPS 175/GNC 355 configured to receive ADS-B In data from GTX 345:<br/>Transponder detects an internal failure.<br/>Functionality may be unavailable.<br/>Possible causes:<br/>• 1090ES ADS-B Out failure<br/>• Transponder failure<br/>• Communication with the transponder is lost | Contact dealer for service.                            |
| Transponder is operating in ground test mode. | For GNX 375, or GPS 175/GNC 355 configured to receive ADS-B In data from GTX 345:<br/>Transponder is being forced airborne for ground test.                                                                                                                                                         | Cycle power to the GNX 375 once ground test completes. |
| ADS-B is not transmitting position.           | For GNX 375:<br/>Transponder is not receiving a valid GPS position.                                                                                                                                                                                                                                 | If the problem persists, contact dealer for service.   |
| \<Unit> demo mode.                            | The unit is in demo mode. Do not use for navigation.                                                                                                                                                                                                                                                | Power cycle the unit to exit demo mode.                |
| LRU Software mismatch.                        | Crossfill function is on but not working due to a software mismatch.                                                                                                                                                                                                                                | Contact dealer for software update.                    |
| Crossfill is inoperative.                     | The unit lost communication with the other crossfill navigator.<br/>One navigator requires service.                                                                                                                                                                                                 | Contact dealer for service.                            |
