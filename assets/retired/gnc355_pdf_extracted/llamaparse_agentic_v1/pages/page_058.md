## COM Standby Control Panel

VHF COM transceiver controls are accessible via the selectable standby (STBY) frequency window. This control resides in the upper right corner of the display.

| \*\*6\*\* | COM Volume Access Key    |           |           |
| \*\*7\*\* | Data Entry Keys          |           |           |


From the COM Standby control panel you can:
- Specify a standby frequency
- Swap active and standby frequency values
- Enable monitor mode
- Access radio volume controls
