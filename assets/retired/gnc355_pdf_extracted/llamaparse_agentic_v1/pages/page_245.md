**Hazard Awareness**

**Graphical AIRMETs**

Graphical AIRMETs (G-AIRMETs) display more weather phenomena than textual AIRMETs, while eliminating the need to interpret raw text. Updates occur four times daily.

Select any G-AIRMET line to view details.
