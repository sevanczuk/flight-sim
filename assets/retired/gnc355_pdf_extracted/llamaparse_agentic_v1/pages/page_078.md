This page from the Pilot's Guide provides instructions for setting up and using the transponder (XPDR) data display.

#### **XPDR Setup**

This section explains how to access the transponder setup menu and the available options.

- **Accessing the Menu:**
  - The user must tap the **Menu** button to access the transponder setup options.

- **Setup Options:**
  - The following actions can be performed from the transponder setup menu:
    - Change the display of data.
    - Enable 1090 ES ADS-B Out functionality (if the aircraft is configured for it).
    - Assign a unique flight ID.

- **XPDR Menu Structure:**
  The image displays a hierarchical menu structure for the transponder setup:
  - **XPDR Menu**
    - **Data Field**
      - PRESS ALT
      - FLT ID
    - **ADS-B Out**
    - **Flight ID**
      - Specify flight ID (if configurable)

#### **Displaying Data**

This section describes how to toggle between two data fields on the display.

- **Data Field Toggle:**
  - The user can toggle the data field between **Pressure Altitude** and **Flight ID**.

- **Pressure Altitude Display:**
  - **Visual Example:** A display shows "Pressure ALT: 2297 FT".
  - **Description:** This field displays the current pressure altitude.

- **Flight ID Display:**
  - **Visual Example:** A display shows "Flight ID: FLY4GA".
  - **Description:** This field displays the active Flight ID. It is noted that unless the system is configured, the Flight ID is not editable.
